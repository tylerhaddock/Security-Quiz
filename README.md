# Security+ SY0-701 Quiz App 🛡️

A Python-based multiple choice quiz application built to help study for the CompTIA Security+ SY0-701 certification exam.

## What It Does

- 100 multiple choice questions covering Security+ SY0-701 exam objectives
- Randomizes question order every session so no two runs feel the same
- Tracks your score in real time so you always know where you stand

## Why I Built It

I built this to study for my Security+ certification directly from my Raspberry Pi. Rather than just reading through material passively, I wanted something hands-on that would build real confidence going into the exam — and give me a Python project worth showing.

## Tech Stack

- Python 3

## How To Run It
```bash
git clone https://github.com/tylerhaddock/security_quiz.py
cd security_quiz.py
python security_quiz.py
```
